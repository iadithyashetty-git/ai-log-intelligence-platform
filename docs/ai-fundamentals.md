AI - Artificial Intelligence is the broad field of making computers perform tasks that normally require human intelligence.

    Machine Learning - Machine Learning is a subset of AI where computers learn patterns from data instead of being explicitly programmed.

      Deep Learning - Deep Learning is a subset of ML that uses neural networks.
      
LLM - Large Language Model
      It predicts the next token (word/subword) repeatedly.
      It doesn't "know" facts—it predicts statistically likely continuations based on training.

RAG - Retrieval-Augmented Generation

Instead of relying only on what the model learned during training, we first retrieve relevant documents (your logs, PDFs, manuals, etc.) and then let the model answer using that retrieved context.
